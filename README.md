# UILayoutTools

![layout_toolbar](https://github.com/simplex86/UILayoutTools/blob/main/_Doc/Images/ugui_layout_toolbar.png)

**上对齐**<br />
**左对齐**<br />
**下对齐**<br />
**右对齐**<br />
**水平对齐**<br />
**垂直对齐**<br />
**相同宽度**<br />
**相同高度**<br />
**相同大小**<br />
**水平等边距**<br />
**水平等中心距**<br />
**垂直等边距**<br />
**垂直等中心距**<br />
**关闭工具栏**<br />
